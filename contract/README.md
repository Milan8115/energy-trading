# ⚡ Energy Trading Smart Contract (Soroban)

## 📌 Project Description

This project is a simple **Energy Trading Smart Contract** built using **Soroban** on the Stellar blockchain. It demonstrates how users can trade energy units in a decentralized way without relying on a central authority.

---

## ⚙️ What It Does

* Allows a user (seller) to list energy units for sale
* Allows another user (buyer) to purchase energy units
* Stores all listings securely on the blockchain
* Updates available energy after each transaction

---

## ✨ Features

* 🔹 Create energy listings
* 🔹 Buy energy units
* 🔹 View available listings
* 🔹 Fully decentralized logic
* 🔹 Built using Rust and Soroban

---

## 🛠️ Tech Stack

* Blockchain: Stellar
* Smart Contracts: Soroban
* Language: Rust

---

## 🚀 Use Case

This project can be used for **peer-to-peer energy trading**, such as:

* Selling extra solar power
* Local energy marketplaces
* Microgrid energy sharing
