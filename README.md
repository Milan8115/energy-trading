# ⚡ Energy Trading Smart Contract

🚀 A decentralized energy marketplace built using **Soroban on Stellar**

---

## 📌 Project Description

This project is a **basic energy trading smart contract** that allows users to buy and sell energy units on the blockchain. It demonstrates how decentralized systems can be used to create transparent and trustless marketplaces.

---

## ⚙️ What It Does

* Sellers can list energy units with a price
* Buyers can purchase available energy units
* The contract automatically updates remaining energy
* All data is stored securely on-chain

---

## ✨ Features

* 🔹 Create energy listings
* 🔹 Buy energy units
* 🔹 View listing details
* 🔹 Decentralized logic (no middleman)
* 🔹 Built using Rust + Soroban

---

## 🛠️ Tech Stack

* **Blockchain:** Stellar (Soroban)
* **Language:** Rust
* **Tools:** Soroban CLI, VS Code

## 🧠 How It Works (Simple)

1. Seller creates a listing (energy + price)
2. Buyer selects a listing and buys units
3. Contract updates remaining energy automatically
